# Expense Tracker

A modern, professional expense tracking application with comprehensive financial management features.

## Features

- Multi-wallet management with currency support
- Real-time expense tracking and categorization
- Budget planning with visual alerts
- Advanced analytics and spending insights
- Monthly trend analysis
- Transaction search and filtering
- Data export capabilities
- Responsive design for all devices

## Quick Start

```bash
npm install --legacy-peer-deps
npm start
```

The application will open at `http://localhost:3000`

## Building for Production

```bash
npm run build
```

## Technology Stack

- React 18
- TailwindCSS
- Recharts for analytics
- Radix UI components
- shadcn/ui design system

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
